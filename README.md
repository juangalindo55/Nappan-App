# 🚀 Nappan App
> "Tu bienestar, tu estilo, tu app."

Este es el repositorio central de **Nappan**, una aplicación enfocada en estilo de vida (Coffee & Fitbar) desarrollada con una metodología de **Vibe Coding** utilizando **Claude Code**.

## 🛠️ Tecnologías
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
- **IA Partner:** Claude Code (Anthropic).
- **Control de Versiones:** Git & GitHub.
